# Week 2 Retro

## What I learned
How to use filter, map, and forEach to transform and analyze arrays.How to structure small utility functions like getAdults, getNames, and countByAge.How to manage state in a small app using an array (todos) and re-render the UI after every change.How to keep UI and logic separated into small, focused functions (render, addTodo, toggleTodo, etc.).

## What was hardest
Understanding how state updates (like reassigning todos) affect rendering.Remembering to re-render the UI after every state change.Handling async logic correctly and making sure data is loaded before rendering.

## Biggest mistake I made
Forgetting to clear the list before re-rendering.Not handling edge cases (like empty input or failed fetch) at first.Overcomplicating simple logic before realizing a simpler array method could solve it.

## What I will do differently next week
 Plan the state structure before writing the UI logic.Break problems into smaller steps before coding.Test each function individually before connecting everything together.Spend more time reviewing fundamentals to strengthen confidence with JavaScript basics.
