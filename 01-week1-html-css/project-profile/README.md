## Goal
Build a single responsive page (no JS) using **semantic HTML** + **Flexbox** + **Grid**.

## Requirements
### Sections
1) Header (logo left, nav links, CTA button)
2) Hero (h1 + intro + button)
3) About (2 columns on desktop, stacked on mobile)
4) Projects (6 cards in responsive grid)
5) Contact (real form with labels + required fields)
6) Footer

### Breakpoints
- 600px (tablet)
- 900px (desktop)

## Deliverables
- `index.html`
- `styles.css`
- 2 screenshots in PR: mobile + desktop

## Checklist
- [ ] Exactly one `h1`
- [ ] No horizontal scroll on mobile
- [ ] Labels connected to inputs
- [ ] Flexbox used for header
- [ ] Grid used for project cards
