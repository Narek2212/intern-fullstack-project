# Intern Full‑Stack Bootcamp (Self‑Learning)

A day‑by‑day curriculum for learning **front‑end development with React** (and prep for full‑stack later).

## Daily workflow
1) Open today’s `README.md`  
2) Read **Learn** + open the **Resources**  
3) Do the **Tasks**  
4) Complete **Self‑check**  
5) Commit → push → PR

## Progress tracker
- [ ] Day 00: Setup
- [ ] Week 1: HTML/CSS foundations + Profile page
- [ ] Week 2: JavaScript foundations + DOM + Fetch + To‑Do
- [ ] Week 3: React fundamentals + hooks + data fetching
- [ ] Week 4: Routing + shared state + forms + testing
- [ ] Capstone: ship a small product app
