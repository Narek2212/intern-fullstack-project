# Capstone (1–2 weeks)

## Goal
Ship a small product-like React app and deploy it.

## Pick one
- Task tracker (CRUD + filters)
- Notes app (CRUD + search)
- Movie explorer (search + favorites)
- Expense tracker (CRUD + categories)

## Requirements
- Routing (3–5 pages)
- API usage (public API or mock server)
- Loading + error states
- Shared state (Context/reducer or small store)
- Forms with validation
- 8–12 tests
- Deployed (Vercel/Netlify)
- README with screenshots + how to run

## Plan
1) Wireframe + routes list
2) Data model
3) Core flows
4) UI polish + responsiveness
5) Tests + deployment
